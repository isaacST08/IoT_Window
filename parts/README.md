# Parts List

- 1x Raspberry Pi Pico W.
- 1x 42-40 Stepper Motor.
- 1x BigTreeTech TMC2209v1.3 Stepper Motor Driver.
- 2x Limit Switches (Mine are from a old Ender 3 V2 3D printer).
- 1x Acme threaded rod that is the slightly longer (or more) than the desired
  travel length of your window.
- 1x Matching acme threaded rod attachment nut.
- 1x Threaded rod to stepper motor coupler.
- 2x Threaded rod bearing mounts.
- 1x 30x10 standard bread board.
- 2x 150 ohm resistors.
- 1x 100 ohm resistor.
- 1x L7805CV voltage regulator.
- 1x Red LED.
- 1x Yellow LED.
- 1x Blue LED.
- 1x 100 microfarad, 30V+ capacitor.
- 1x Power supply, 12V to 28V (I'm using 12V).
- 1x Female barrel plug that matches power supply.
- 3x M4 * 20 Bolts.
- 3x M4 Nuts.
- 4x M3 * 8 Bolts.
- 4x M3 * 12 Bolts.
- 4x M3 * 20 Bolts.
- 4x M3 Nuts.
- Plenty of wires.
  - If only using a breadboard and following the breadboard wiring diagram, the
    low profile style breadboard wires will be needed.

# Tools

- A 3D printer or access to one that can print PETG.
